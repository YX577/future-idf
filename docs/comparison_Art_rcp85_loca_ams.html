Differences between CMU CF and Art's CF (LOCA)

The interactive plots below show the difference (CMU - Art) in CF calculated using different methods. Each dot corresponds to a station location, and it is color-filled with the value of the difference. You can hover to see the station's GHCN ID and the respective value of the difference.


Choose a return period from the drop-down menu below to see the differences in other return periods. You can zoom-in the map by choosing the "Wheel Zoom tool" on the top right (next to each color bar) of each individual figure.
